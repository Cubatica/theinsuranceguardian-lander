# Hero Background Image

## Required Image

Place a car background image in this directory:

- **File name**: `hero-car.jpg` (or `hero-car.png`)
- **Recommended size**: 1920x1080px or larger
- **Format**: JPG or PNG
- **Content**: Professional car image (modern car, highway, or automotive scene)

## Image Suggestions

The hero section will display this image with:
- 20% opacity overlay
- Blue gradient overlay
- Pattern texture
- Dark overlay for text readability

## Free Image Resources

- **Unsplash**: https://unsplash.com/s/photos/car-highway
- **Pexels**: https://www.pexels.com/search/car/
- **Pixabay**: https://pixabay.com/images/search/car/

## Current Status

If no image is found, the hero will display with just the gradient background (which still looks professional).
